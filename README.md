# Ball-Trap Master Pro — V1.3

Application de coaching ball-trap pour entraîneurs FFBT.

## Fonctionnalités
- Gestion multi-élèves avec profils (Débutant / Confirmé / Expert)
- Coaching temps réel : FU, DTL, TRAP 1, PCH, CS
- Grille 25 tirs avec analyse par zone
- Phrases coaching pseudo-IA (hors-ligne, aucune API)
- Historique par élève avec arborescence
- Statistiques avancées + radar par discipline
- Export PDF bilans détaillés
- Météo auto + vent sur le pas de tir
- 3 thèmes : clair, sombre, plein soleil
- PWA installable (hors-ligne)

## Installation
1. Ouvrir `index.html` dans un navigateur ou déployer sur GitHub Pages
2. L'application fonctionne 100% hors-ligne après le 1er chargement
